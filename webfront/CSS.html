                          # CSS

                          CSS：层叠样式表格

                          作用：1.装饰效果 2.网页布局

                          1.查找元素：提供了大量的选择器

                          2.控制元素：提供了很多元素控制效果

                          3.布局：根据效果图将元素放置在合适的位置上

                          目标：给你一个设计图，然后转化成一个网页HTML+CSS PSD=>静态页

                          # CSS使用

                          1.内联方式（行内样式表）:直接在元素标签中写style属性，设置相应的样式效果

                          <标签名 style='属性:值;属性:值;'></标签名>

                          2.内部样式表：写在页面的style标签中

                          <head>
    <style>
        #id{
            margin: 100px auto;		/* 外边距 */
            padding: 10px;			/* 内边距 */
            width: 800px;			/* 宽 */
            height: 400px;			/* 高 */
            background-color: #096;	/* 背景颜色 */
            text-align: center;		/* 文本居中 */
        }
	</style>
</head>

                          3.外部引用（外部样式表）:把CSS代码单独放在一个CSS文件中，然后在网页中用link标签进行引用-->开发中使用的模式

                          link rel="stylesheet" href="路径">

                          # CSS选择器

                          # 基本选择器

                          # 1.通配符选择器（*）

                          查找页面中的所有元素

                          *{
    属性:值;
    属性:值;
}

                          # 2.标签选择器（tag）

                          通过元素的标签名称进行查找，一般是用来清除样式/设置网页的默认样式

                          标签名{
	属性:值;
	属性:值;
}

                          # 3.id选择器（#id）

                          通过元素的id属性进行查找

                          <标签名 id='id名'></标签名>

#id名{
	属性:值;
	属性:值;
}

                          # 4.类选择器（.class）

                          通过class属性进行查找

                          <标签名 class='类名'></标签名>

.class名{
	属性:值;
	属性:值;
}

                          # 后代选择器（空格）和子代选择器（>）

                          后代选择器：找到所有的后代元素包括子元素

                          .class名 p{
	属性:值;
	属性:值;
}

                          子代选择器：只查找子元素，不会去查找后代元素（孙子和曾孙都不选）

                          .class名>p{
	属性:值;
	属性:值;
}

                          # 兄弟选择器（+）和同胞选择器（~）

                          <ul>
    <li>0</li>
    <li id="ll">1</li>
    <li>2</li>
    <a>aaa</a>
	<li>3</li>
</ul>

                          兄弟选择器：查找与自己同辈，且向下紧邻的元素

                          <!--选中的是2-->
<style>
    #ll+li{
        属性:值;
    }
</style>

                          同胞选择器：查找与自己同辈，且向下的所有元素

                          <!--选中的是2，3-->
<style>
    #ll~li{
        属性:值;
    }
</style>

                          # 交集选择器（无符号）和并集选择器（,）

                          <ul>
    <li class="c1">0</li>
    <li id="ll">1</li>
    <li class="c1">2</li>
    <a>aaa</a>
    <li>3</li>
</ul>

                          交集选择器：选中两元素相交的部分

                          <!--选中的是0，2-->
<style>
    ul li.c1{
        color: red;
    }
</style>

                          并集选择器：并的元素都选中，一次性选择多个元素

                          <!--选中的是0,1,2,aaa,3-->
<style>
    ul li,a{
    	color: red;
    }
</style>

                          # 伪类（:）选择器和伪元素（::）选择器

                          伪类选择器：不是真实存在的class，但是CSS有认为有这个class

                          元素:link{}
/* 当a标签未被访问过时，触发样式 */

元素:active{}
/* 当按下元素不放开时，触发样式 */

元素:visited{}
/*  指的是访问过的伪类状态,当a标签被访问过时，触发样式 */
注意：由于涉及用户的隐私，所以该伪类只能更改颜色 不能更改其他样式效果

元素:hover{}
/* 鼠标悬停在元素上的伪类状态 */

                          伪元素选择器：元素在代码中不存在，但是在实际效果中存在，不能被选中，只能看不能用，性能较好

                          元素::after {
	content: '》';
}/* 在元素最后面添加伪元素 */


元素::first-line {
	color: red;
}/* 在元素的第一行添加效果 */


元素::before {
	content: "《"
}/* 在元素最前面添加伪元素 */


元素::selection {
	background-color: red;
	color: black;
}/* 选中元素时的样式 */

                          # 属性选择器（[属性]）

                          根据元素的属性值来获取元素

                          [属性] 选中有该属性的元素
[属性=属性值]	选中属性=属性值的元素
[属性|=x]	匹配属性值开头为x-的元素
[属性~=x]	匹配属性值开头为x 的元素
[属性^=x]	匹配属性值以x开头的元素
[属性$=x]	匹配属性值以x结尾的元素
[属性*=x]	匹配属性值包含x的元素

                          # CSS3选择器

                          nth-child选择器

                          元素:nth-child(参数){}			正着数
元素:nth-last-child(参数){}		倒着数
元素:nth-first-child{}		第一个
/* 条件:1.当前元素要有父元素
		2.父元素要有若干个子元素
		3.第n个位置上的元素必须和当前元素的类型相同*/


元素:nth-of-type(参数){}		正着数
元素:nth-last-of-type(参数){}	倒着数
元素:nth-first-of-type{}		第一个
/* 条件:1.当前元素要有父元素
		2.父元素要有若干个子元素*/

/* 参数：1.整数(写几就是第几个子元素，默认是从1开始的)
		2.表达式(2n+1奇数、2n偶数)
		3.odd奇数、even偶数*/

:not(){}	不包含

                          在CSS选择器中，各类不同的选择器是存在权重的概念

                          !important 						最高权重，任何选择器的权重都不能高于该关键词(谨慎使用)
内联样式						1000	p4
id选择器						 100	 p3
类选择器						10		p2
标签选择器					    1	   p1
通配符选择器/继承样式				 0		p0

选择器权重是可以累加的
注意：选择器虽然可以累加，但是不能超过上一级的权重值

                          # 单位

                          px:像素	像素代表的是你电脑屏幕上的一个点

%:百分比	元素相对于其父元素的大小比例，比较优秀单位，但是用起来麻烦    响应式（在不同设备上，大小是等比例缩放）

em:其父元素的1个字体大小
rem:相对于html中一个字体大小    适用在移动端布局的时候
注意:em和rem区别点在于参考对象是不同的，em参考父元素的字体大小，1em=1个父元素字体大小，1rem=1个html中字体的大小

deg:度 0-360 控制元素角度

vw(viewwidth):视口宽度   100vw
问题：如果使用像素布局，那么无法做到适应所有的设备 例如开发者的屏幕宽度是960  而使用者宽度是1000 那么就会造成开发者看到的页面和使用者的页面是不同的
使用px进行布局，网页是非响应式
解决方案：1.使用%布局 不现实
        2.使用vw进行布局

提示：一般情况下，我们默认iphone6物理尺寸为黄金尺寸 375   
750px设计图为2倍图		1px=100vw/750px=0.13333333vw	计算出1px等于多少vw
1125px设计图为3倍图		1px=100vw/1125px=0.0888888vw	计算出1px等于多少vw
先看给的pad图的宽度多少，然后让100vw/宽度即可计算出在对应设备上1px对应的vw

                          margin:10px	上右下左各10个像素
margin:10px 20px   设置垂直外边距上下为10,水平外边距左右为20px
margin: 10px 8px 5px 2px; 设置上右下左分别为10,8,5,2
margin-bottom: 100px; 设置某一方向的外边距
margin-top: 100px; 设置某一方向的外边距
margin-left: 100px; 设置某一方向的外边距
margin-right: 100px; 设置某一方向的外边距

                          外边距产生的问题：

                          1.相邻元素垂直外边距重叠时，设置垂直外边距时取最大值而不是叠加

                          2.父子元素外边距重叠时，设置子元素的垂直外边距时，父元素跟着改变。

                          解决：1)给父元素设置1px的上部内边距 （推荐）

                          ​ 2)设置父元素内边距

                          # 浮动（float）

                          float:left	左浮
	  right	右浮

浮动特点：
	1.元素浮动后，该元素会脱离文档流
	2.元素脱离文档流后，层级要高于原来的层级  显示时优先显示高层级元素
	3.当元素浮动时，元素属性display会变为inline-block

补充：
	BFC:block format content 块级格式上下文	开启bfc模式后，会变成行内块级元素，不会影响其他的元素，CSS中没有专门属性开启bfc模式
开启方式：
	1.让元素浮动
	2.更改display:inline-block
	3.overflow:非visiable
	4.开启绝对定位

                          浮动产生的问题：当子元素浮动后，会产生父元素高度坍塌。

                          解决方案：

                          ​ 1.给父元素设置高度 （解决不了问题）

                          ​ 2.父元素尾部设置一个块级元素，利用clear:both属性来清除浮动影响 缺点：莫名其妙留下一个div

                          ​ 3.给父元素添加一个伪元素（最优）

                          .clearfix::after {
    /* 没有内容 */
    content: '';
    /* 设置块级元素 */
    display: block;
    /* 兼容性属性 */
    zoom: 1;
    /* 清除浮动影响 */
    clear: both;
}

子元素浮动就给父元素加上clearfix类

                          # 再谈块级元素，行内元素，行内块级元素

                          # 块级元素（block）

                          特点：
	1.独占一行
	2.可以设置宽高、内外边距、边框

                          # 行内元素（inline）

                          特点：
	1.宽高：不能设置宽高
	2.外边距：只有水平方向的外边距，没有垂直方向的外边距
	3.内边距：四个方向都有	注意：垂直方向和其他元素紧挨时，内边距会失效
	4.边框：可以设置边框
	5.不独占一行

                          # 行内块级元素（inline-block）

                          特点：不独占一行，其余和块级元素特点一样

                          # 三种元素之间的转化

                          display:block	块级元素
		inline	行内元素
		inline-block	行内块级元素
		none	隐藏

                          # 定位（position）

                          # 相对定位（relative）

                          position:relative
top:10px
left:10px

相对于原来位置的定位方式
设置元素的偏移量left top bottom right 一般只使用top和left

z-index:数值	设置定位后的元素的层级
注意：必须两个元素都开启定位后才能使用

特点：
	1.相对定位不改变元素的性质
	2.相对定位原来的位置时保留的
	3.必须要设置偏移量
	4.定位会高于文档流元素的一个层级

                          # 绝对定位（absolute）

                          子绝父相

                          position:asolute

相当于自己祖先元素第一个开启定位(什么定位都可以)的元素进行定位，如果祖先元素中没有开启定位的，则根据body进行绝对定位

特点：
	1.绝对定位不保留原来的位置
	2.绝对定位会改变元素的性质
	3.脱离文档流

                          # 固定定位（fixed）

                          position:fixed

根据浏览器的位置定位

特点：
	1.固定定位一直在浏览器某个位置上固定
	2.脱离文档流
	3.元素性质会发生改变
	
固定和绝对的区别：
	一个是根据浏览器的位置定位(下拉后位置也不变)，一个是根据父级开启定位的元素来定位的

                          # 背景（background）

                          设置背景颜色

                          background-color:英文单词/三原色/十六进制

                          设置背景图片

                          background-image:url(路径)

设置图片重复方式(当图片小于盒子的时候)
background-repeat:repeat(默认值)	水平方向和垂直方向都重复
				  no-repeat	不重复
				  repeat-x	水平方向重复
				  repeat-y	垂直方向重复

设置背景图片大小
background-size:300px(宽) 200px(高)

设置背景图片的位置
background-position:垂直方向上(top/center/bottom) 水平方向上(left/center/right)
					单位进行移动(200px 100px)

                          # 过度

                          transition-property:all/margin-left;	过度的属性，如果所有属性都要过度就写all
transition-duration:2s/2000ms;			过度的时间，可以是秒也可以是毫秒
transition-timing-function:steps(5);	过度函数，分5步走
						   ease-out;	先快后慢
						   ease-in;		先慢后快

transition:all 3s steps(5);	一键设置

注意：1.只有有单位的属性可以进行过度
	 2.必须要有触发条件
	 3.只能执行一次，不可以一直执行

                          # 动画

                          动画实际上是过度的升级版，动画可以自动触发，并且可以设置执行的次数。

                          步骤：
	1.提供关键帧		就是初始位置和结束位置
	2.动画执行时间
	3.动画执行方式
	4.动画的执行次数

/* 关键帧 */
@keyframes run/*关键帧名称*/ {
    to {
        margin-left: 700px;	/* 结束时的属性 */
    }

    from {
        margin-left: 0px;	/* 开始时的属性 */

    }
}


animation-name:run;		关键帧名称;
animation-duration:2s;	动画执行时间
animation-iteration-count:n(执行n次)/infinite(无限执行);

                          # 弹性盒布局（flex）

                          优点：非常的简单 缺点：只适用于移动端；新浏览器

                          弹性容器：只要开启弹性布局属性的元素就是弹性容器

                          /* 弹性元素的布局方向 */
flex-direction:row(默认)		行从左向右
			   row-reverse	 行反转从右向左
			   column		 列从上到下
			   column-reverse列从下到上

/* 设置弹性元素换行方案 */
flex-wrap:nowrap(默认)	不进行换行，自行对元素进行压缩
		  wrap			 不进行压缩元素，当宽度不够时换行
		  wrap-reverse	 不进行压缩，当宽度不够时换行，然后上下置换

/* 设置弹性容器中弹性元素排布位置 */
justify-content:left/start(默认) 靠左
				center			居中
				right/end		靠右
				space-around	剩余控件均等分后环绕在弹性元素周围
				space-between	保证首尾有弹性元素排布，其余进行等分
				space-evenly	剩余的控件每个弹性元素间隔相同

                          弹性元素：弹性容器中的元素就是弹性元素

                          /* 设置弹性元素的位置先后 */
order:数字	数字越小 排列约靠前(考虑主轴方向)

/* 弹性元素成长属性 */
flex-grow:数字	弹性元素会根据该属性将父元素未使用的部分进行等比例拆分 数值越大 占位越多

/* 弹性元素压缩属性 */
flex-shrink:数字	数值越大，压缩越狠
