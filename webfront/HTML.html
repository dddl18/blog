        # 块级元素（block）

        常用的块级元素：ul,li,table,tr,th,p,h1~h6,form,div

        特性：

        1.独占一行

        2.块元素有宽度，高度，内/外边距属性，这些都可以自己设置

        3.块元素的宽度默认是浏览器的100%，高度为内容高度

        4.块元素可以包裹行内元素/其他块元素

        # 内联（行内）元素（inline）

        常用的行内元素：a,span,input,img		b,strong,i,em,u,ins

        特性：

        1.不独占一行，默认都是放在一起

        2.行内元素的宽度就是自身内容的宽度

        3.行内元素设置宽高，外/内边距是没有意义的，它不具备结构能力

        4.行内元素从规范性来说，是不能放块元素，只能放其他行内元素

        注意：

        1.a标签内可以放块元素，因为它和用户体验相关，所以a标签可以包裹块元素

        2.img虽然可以设置宽高，但它不是块元素，因为宽高是它自身的属性

        # HTML5

        HTML5是HTML最新的修订版本，2014年10月由万维网联盟（W3C）完成标准制定。

        HTML5的设计目的是为了在移动设备上支持多媒体。

        HTML5相较于HTML4的新特性：

        1.加入了绘画canvas元素 svg矢量绘画

        2.添加了媒体播放video和audio元素

        3.添加了本地离线存储的功能

        4.添加了新的语义标签，例如：article、footer、header、nav、section

        5.新的表单控件,例如：calendar、date、time、email...

        6.移动端功能（地理定位等）

        # 本地存储

        使用HTML5可以在本地存储用户的浏览数据。

        以往存储数据通常都会保存在cookie上，但是cookie存取效率较低。HTML5中提出了2种全新的存储，分别是：localStorage和sessionStorage

        • localStorage - 用于长久保存整个网站的数据，保存的数据没有过期时间，直到手动去除。
        • sessionStorage - 用于临时保存同一窗口(或标签页)的数据，在关闭窗口或标签页之后将会删除这些数据。

        存储：

        localStorage.setItem(key,value);

        读取：

        localStorage.getItem(key);

        移除：

        localStorage.removeItem("mydate");

        <details> 定义用户可查看或隐藏的额外细节。

        # 地理定位

        <p id="demo">点击按钮获取您当前坐标（可能需要比较长的时间获取）：</p>
<button onclick="getLocation()">点我</button>
<script>
var x=document.getElementById("demo");
function getLocation()
{
	if (navigator.geolocation)
	{
		navigator.geolocation.getCurrentPosition(showPosition);
	}
	else
	{
		x.innerHTML="该浏览器不支持获取地理位置。";
	}
}

function showPosition(position)
{
	x.innerHTML="纬度: " + position.coords.latitude + 
	"<br>经度: " + position.coords.longitude;	
}
</script>
