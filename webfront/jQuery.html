# 事件

就绪事件

写法一：
$(document).ready(function(){
	代码
});

写法二：
$().ready(function(){
	代码
});

写法三(推荐)：
$(function(){
	代码
});

监视事件

//监视
$("#p1").click(function () {
    console.log(1);
})

//进行批量监视
$("p").click(function () {
    //点击的元素就是this，但是是DOM对象，需要先转化为jQuery对象再操作
    $(this).css("color", "red");
})

//原本的addEventListener
$("#p1").on("click",function(){
    console.log(1);
});

# 文档处理

append()：在元素里面的尾部添加东西

prepend():在元素里面的首部添加东西

<p>I would like to say: </p>
<script>
	$("p").append("<b>Hello</b>");
    $("p").prepend("<b>小小</b>");
</script>

<!--[ <p><b>小小</b>I would like to say: <b>Hello</b></p> ]-->

after()：在元素后面添加元素（同级）

before()：在元素前面添加元素（同级）

<p>I would like to say: </p>
<script>
	$("p").after("<b>Hello</b>");
    $("p").before("<b>小小</b>");
</script>

<!--
	<b>小小</b>
	<p>I would like to say: </p>
	<b>Hello</b>
-->

replaceAll()：将元素代替选中的元素

<p>Hello</p>
<p>cruel</p>
<p>World</p>
<script>
	$("<b>Paragraph. </b>").replaceAll("p");
</script>

<!--
	<b>Paragraph. </b>
	<b>Paragraph. </b>
	<b>Paragraph. </b>
-->

empty()：清除元素的所有子元素

<p>Hello, <span>Person</span> <a href="#">and person</a></p>
<script>
	$("p").empty();
</script>

<!--
	<p></p>
-->

remove()：删除匹配的元素

detach()：删除匹配的元素

<p>Hello</p> how are <p>you?</p>
<script>
	$("p").remove();
</script>

<!--
	how are
-->

# 筛选

hasClass(class)：匹配元素是否有指定类名

<div class="protected" id="d1"></div><div></div>
<script>
	var res = $('#d1').hasClass("protected");
    console.log(res)
</script>

<!--
	true
-->

slice(start,end)：选取第start个开始到end结束之间的元素

<p>Hello</p><p>cruel</p><p>World</p>
<script>
	$("p").slice(1, 2).css('color','red');
</script>

<!--
	cruel变红
-->

children()：查找匹配元素的所有子元素（不包括后代） 只找子代

find()：从当前匹配元素开始搜索匹配表达式的后代元素 找后代

<p>Hello</p><div><span>Hello Again</span></div><p>And Again</p>
<script>
	$("div").children();
</script>

<!--
	[ <span>Hello Again</span> ]
-->

<p><span>Hello</span>, how are you?</p>
<script>
	$("p").find("span");
</script>

<!--
	[ <span>Hello</span> ]
-->

closest()：查找匹配元素的符合表达式的第一个祖先/父元素，支持查找多个元素（必须写表达式参数）

parent()：查找匹配元素的父元素，不支持查找多个元素（只能查找父亲，不能查找祖先，不写参数就是找父亲元素）

parents()：查找匹配元素的所有符合表达式的祖先元素

<ul><li>1</li><li>2</li></ul>
<script>
    $("li:first").closest(['ul','body']);
	$("li:first").parent('ul');
</script>

<!--
	[ ul,body ]
-->
<!--
	[ ul ]
-->
